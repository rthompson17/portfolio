Cosmos - Python / Django Application

Cosmos

  • Technologies:: Python3, Django, JavaScript HTML, CSS, PSQL, Materialize
  • Client: General Assembly Software Engineering Immersive
  • Project date: April, 2022
  • Project URL: https://github.com/rthompson17/cosmos-app

Cosmos is a full stack application where I served as a Database Manager and a Designer on a team of four using, Python and the Django framework. Our team wanted to build an application that could help people find cosmic events happening near them while creating an opportunity to meet other astrology enthusiasts, gathering with them at user-generated “viewing parties.”

As a team, we put a lot of planning into our application's design and functionality. Througout the process, we toggled between solo, pair and mob programming, and now that our General Assembly immersive program is over, we continue to work together as a team to further-enhance this app. Learn more about our project on GitHub and visit the site to follow our progress.
